Open Source C# ASP.NET Document Portal - DocFlow Readme Doc:

DocFlow is an application that demonstrates how to create a powerful Document Portal using the Microsoft .NET platform, Microsoft SQL Server and Microsoft Indexing Service. DocFlow provides documents storage and management, content search and role-based security to ensure that only right people access you sensitive data. DocFlow supports many document formats including Microsoft Office documents. The latest version of DocFlow, along with the source code and documentation will always be available on:
